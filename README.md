# How-to-learn-408

## 前言

### 什么是 408 ？

408 是中国硕士研究生入学考试计算机全国统考的科目代码。

### 408 有哪些内容？

408 的内容包括：数据结构、计算机组成原理、操作系统、计算机网络。

### 学习 408 的前提

- 掌握一门计算机编程语言，本项目建议掌握 C 或 C++。
